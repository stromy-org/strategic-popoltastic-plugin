# Proposal Archetypes

Five proposal archetypes that pre-configure section selection, tone, depth, and structure. Use during intake (Phase 2) to shortcut configuration for common B2B scenarios.

## Archetype Definitions

### Corporate

**Target clients**: Enterprise, government, large organizations with formal procurement.
**Tone**: Formal, structured, trust-building. Evidence-heavy, process-oriented.
**Depth**: Full (15-25 pages).
**Sections**: All 15 from the standard skeleton.
**Pricing model**: Detailed breakdown by phase with payment milestones.

Key emphasis:
- Methodology rigor and governance
- Team credentials and organizational credibility
- Risk management and compliance
- Formal terms and conditions

### Consultant

**Target clients**: Advisory, professional services, C-suite strategic engagements.
**Tone**: Consultative, analytical, solution-focused. Leads with diagnosis before prescription.
**Depth**: Standard (8-15 pages).
**Sections**: 12 — skip Appendices and Risk Management (fold key assumptions into Pricing).
**Pricing model**: Tiered engagement options (see Tiered Pricing below).

Key emphasis:
- Situation analysis with current state / desired outcome arc
- Strategic priorities and implementation considerations
- 2-3 engagement options with KPIs per tier
- Expected outcomes framework

### Entrepreneur

**Target clients**: Startups, SMBs, founders, fast-moving businesses.
**Tone**: Bold, direct, action-oriented. Short sentences, no fluff.
**Depth**: Brief (4-8 pages).
**Sections**: 7 core — Cover, Executive Summary, Client Context (as "Problem"), Approach (as "Solution"), Deliverables (as "What You Get"), Pricing (single package), Next Steps.
**Pricing model**: Single package, transparent.

Key emphasis:
- Speed and time-to-value
- Concrete deliverables with timeline
- Direct call to action
- Minimal process overhead

### Creative

**Target clients**: Agencies, design firms, marketing/branding services.
**Tone**: Inspirational, collaborative, visually modern. Narrative-driven.
**Depth**: Standard (8-15 pages).
**Sections**: 10 — Cover, Executive Summary, Client Context (as "Vision"), Approach (with philosophy statement), Deliverables, Relevant Experience (case-study-heavy), Team, Pricing (with optional add-ons), Next Steps.
**Pricing model**: Base package with add-on modules.

Key emphasis:
- Portfolio and case studies as primary proof
- Creative philosophy and values alignment
- Visual presentation quality
- Collaborative language ("we" over "I")

### Minimal

**Target clients**: Freelancers, quick quotes, repeat clients, fixed-scope projects.
**Tone**: Clean, efficient, no-fluff. Purely factual.
**Depth**: Brief (2-4 pages).
**Sections**: 6 essential — Cover, Scope (combines Context + Approach), Deliverables, Timeline, Pricing, Terms/Acceptance.
**Pricing model**: Single line item or simple table.

Key emphasis:
- Fast turnaround
- Clear scope boundaries
- Direct acceptance mechanism
- No narrative beyond essentials

## Default Pairing Table

| Client Type | Archetype | Depth | Typical Sections |
|-------------|-----------|-------|-----------------|
| Enterprise / Government | Corporate | Full (15-25 pages) | All 15 |
| Advisory / Strategic | Consultant | Standard (8-15 pages) | 12 (skip Appendices, Risk Mgmt) |
| Startup / SMB | Entrepreneur | Brief (4-8 pages) | 7 core sections |
| Agency / Design | Creative | Standard (8-15 pages) | 10 (portfolio emphasis) |
| Quick quote / Repeat | Minimal | Brief (2-4 pages) | 6 essential sections |

## Tone Presets

Each archetype sets default values for the three tone dimensions defined in the main SKILL.md:

| Archetype | Formality | Audience | Stance |
|-----------|-----------|----------|--------|
| Corporate | Formal | Executive / mixed panel | Assertive |
| Consultant | Formal | Executive | Consultative |
| Entrepreneur | Conversational | Technical / founder | Assertive |
| Creative | Conversational | Executive / creative | Consultative |
| Minimal | Formal (terse) | Technical | Neutral |

Users can override any default during Phase 3 (Strategy).

## Tiered Pricing Guidance

The **Consultant** archetype defaults to 2-3 engagement options using the Good/Better/Best pattern:

| Tier | Label | Scope | Purpose |
|------|-------|-------|---------|
| Option A | Core | Essential scope, lowest investment | Anchor — establishes minimum viable engagement |
| Option B | Recommended | Full scope with key additions | Target — most clients choose the middle option (compromise effect) |
| Option C | Premium | Comprehensive scope with strategic extras | Aspirational — reframes Option B as reasonable |

**Presentation tips:**
- Present the highest option first to anchor price expectations
- Mark Option B as "Recommended" explicitly
- Include KPIs or expected outcomes per tier where possible
- Frame as "investment" not "cost" — include expected ROI
- Show a comparison table for easy side-by-side evaluation

Other archetypes can adopt tiered pricing when appropriate — it is not exclusive to Consultant.
