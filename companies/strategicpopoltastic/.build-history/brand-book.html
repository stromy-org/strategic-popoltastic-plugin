05 / Logo Rules

Logo Usage Rules

Protect the integrity of the mark. These rules ensure consistency across all applications.

Clear Space

Minimum clear space equals the height of the letter "P" in Popoltastic, applied to all four sides. No other elements may intrude into this zone.

P height P height P height P height
Color by Background
Background "STRATEGIC" Slash "Popoltastic"
Dark (#0A1628) #8B949E #7FB3D3 #FFFFFF
Light (#F8F9FA) #6E7681 #5A9AB5 #0A1628
Brand (#1B3A5C) rgba(255,255,255,0.5) #7FB3D3 #FFFFFF
Minimum Size

Never render the logo at less than 120px wide. Below this threshold, use the icon mark instead.

Don'ts
✗ Never rotate the logo
✗ Never change the slash angle
✗ Never use the slash in a non-accent color
✗ Never stack without the slash
✗ Never use below 120px wide

Weight Restrictions

Aa

Headlines only. Use sparingly.

Pairing Rules
Syne is display-only. Never use for body text or UI elements.
Space Mono is data-only. Use for numbers, labels, metadata. Never for paragraphs.
Work Sans carries everything else: body, UI, navigation, captions.
Never use more than two weights on a single slide or page section.
10 / The Edge

Brand Motif

The diagonal ice blue slash from the logo, abstracted into a repeatable visual system. 2.5px line at -8deg. The Edge is structure. The Chevron (>) is navigation.

Edge Line

Inline accent via ::before pseudo-element. Used next to headings and labels.

Edge Divider

Full-width hairline with slash at the start. Section separators.

Edge Accent (Short)

Standalone short slash. Bullet points and list markers.

Edge Double

Two parallel slashes, second at 40% opacity. Emphasis and section openers.

Edge Fade

Gradient fade-out. Transitional elements and decorative endings.

Edge Corner

L-shaped slash at corner. Card accents and frame markers.

CSS Implementation
/* Brand motif tokens */
--motif-color: var(--brand-accent);  /* #7FB3D3 */
--motif-weight: 2.5px;
--motif-angle: -8deg;

/* Edge accent — standalone slash */
.edge-accent {
  display: inline-block;
  width: var(--motif-weight);
  height: 24px;
  background: var(--motif-color);
  transform: rotate(var(--motif-angle));
}
11 / Voice

Voice & Tone

How Strategic Popoltastic sounds. Short. Direct. Backed by evidence. A touch of personality.

Writing Principles
Short, declarative sentences. Get to the point.
Active voice, present tense. "We deliver" not "It has been delivered."
Frameworks named and explained simply. No jargon walls.
Confident assertions backed by evidence. Claim + proof.
A touch of personality in tone. Sharp, never sarcastic.
Do / Don't Examples
Do
"Your market position is eroding. Here's the 3-move fix."
Don't
"While challenges exist, there are also exciting opportunities to explore going forward."
Do
"Revenue grew 23% after the process redesign."
Don't
"Significant improvements were observed across multiple business dimensions."
Do
"This won't work. Here's what will."
Don't
"Our innovative team leveraged cutting-edge methodologies to assess opportunities."
